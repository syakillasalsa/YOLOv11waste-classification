# 🌱 EcoSmart Station

EcoSmart Station is a web-based waste classification system designed to support waste management in railway station environments. This application helps users identify and classify waste into three categories: **organic**, **inorganic**, and **hazardous (B3)**.

## 🚀 Features
- 📷 **Real-time Camera Scan**  
  Users can directly scan waste using their device camera.

- 🖼️ **Gallery Image Upload**  
  Users can upload images from their device gallery for classification.

- ♻️ **Waste Classification**  
  Automatically classifies waste into:
  - Organic
  - Inorganic
  - Hazardous (B3)

- 🌐 **Web-Based Access**  
  Easily accessible via browser without installation.

- 🔗 **QR Code Access**  
  Users can quickly access the system by scanning a QR code.

## 🛠️ Technologies Used
- **Python (Flask)** – Backend framework  
- **YOLOv11 Nano** – Object detection model  
- **HTML, CSS, JavaScript** – Frontend  
- **OpenCV** – Image processing  

## 📌 How It Works
1. User opens the web via QR code or browser  
2. Choose to:
   - Scan using camera, or  
   - Upload image from gallery  
3. System processes the image using YOLOv11 Nano  
4. Result shows the waste category (organic, inorganic, or B3)

## 🎯 Purpose
This project aims to:
- Improve waste sorting awareness in public areas  
- Support smarter waste management systems in stations  
- Utilize AI for real-world environmental solutions
